## Управление

| Клавиша | Действие |
|---|---|
| `→` `Space` | Следующий слайд |
| `←` | Предыдущий слайд |
| `1–9` | Прыгнуть на слайд по номеру |
| `R` | **Показать / скрыть русскую речь** (телепромптер справа) |
| `F` | Полноэкранный режим |
| `Esc` | Закрыть русскую панель |
| клик в правую треть | Следующий |
| клик в левую треть | Предыдущий |

Есть также маленький круг `●` внизу по центру — клик = открыть русскую речь.

## Твои файлы (что нужно подставить)

Положи файлы в папку `media/` с такими именами:

| Файл | Слайд | Что это |
|---|---|---|
| `01_g1_hero.mp4` | 1 | 5–8 сек видео G1 в движении |
| `04_you_and_g1.jpg` | 4 | Фото ты + G1 в лаборатории |
| `09_g1_inspection.jpg` | 9 | G1 на промышленной инспекции |
| `13_portrait.jpg` | 13 | Портрет ты + G1 + медаль |
| `15_qr.png` | 15 | QR-код на твой канал / LinkedIn |

Потом открой `index.html` и найди блоки с классом `media-slot` — замени на `<img>` или `<video>`. Или скажи мне, я подставлю.

## Что нужно заполнить текстом

В `index.html` найти и заменить:
- `{{YOUR NAME}}` → твоё имя (слайды 1 и 15)
- `{{TELEGRAM or LINKEDIN}}` → что вести QR (слайд 15)
- `{{t.me/your-channel}}` → ссылка на канал
- `{{your@email}}` → email
- Год в championship (слайд 1) — сейчас стоит "2025"

## Архитектура

